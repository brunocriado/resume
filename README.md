Requirements
------------

* Arch Linux packages to build the TEX version:

 `pacman -S texlive-bin texlive-core texlive-science texlive-latexextra
 texlive-bibtexextra`

Building and Viewing
--------------------

Compiling:

`make`

Viewing:

`make view-pdf` or `make view-asm`

Cleaning (objects):

`make clean`
